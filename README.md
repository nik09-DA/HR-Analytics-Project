# HR Analytics Project

## Project Overview
This project analyzes employee attrition trends using Excel, MySQL, and Tableau.

## Tools Used
- Excel
- MySQL
- Tableau

## Key Insights
- High attrition observed in Production and IT/IS departments
- Lower salary employees showed higher attrition
- Mid-performing employees had the highest attrition rate
- Employees generally left after 3.5 years

## Recommendations
- Improve salary structure
- Provide better career growth opportunities
- Increase employee engagement initiatives
